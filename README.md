Application in .NET MAUI using the PokeAPI for creating and handling "game" level entities in a standard pokemon format.

This project handles the first 151 pokemon for simplicties sake, and in relation to the re-release of Pokemon Firered and Leafgreen on the Nintendo Eshop.

This works as a example battle-engine for the pokemon franchise, but in MAUI.
